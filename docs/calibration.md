# 캘리브레이션 절차

## 1. 부팅 자동 영점 보정

펌웨어는 매 부팅 시 첫 5초간 ADC 값을 평균하여 오프셋(`zeroOffsetCmH2O`)을 저장한다. 이 구간 동안 기기를 입에 물지 않아야 한다(대기압 측정).

## 2. 수동 재캘리브레이션

- 기기 단독: STANDBY 에서 버튼 **3회 연속** 짧게 누르기
- 앱 경유: `Session Control` → opcode `0x04 ZERO_CALIBRATE`

10초간 측정 후 성공 시 진동 2회.

## 3. 공장 캘리브레이션 (개발자용)

병원용 디지털 압력계(U-tube manometer 등)와 비교하여 `sensitivity` 계수 교정.

1. 시린지로 10, 20, 30 cmH2O 를 기기와 레퍼런스에 동시 인가
2. 각 지점 3회 측정, 기기값과 레퍼런스 비 계산
3. 평균 비를 `config/sensitivity_factor` 로 EEPROM 저장
4. 최종 오차 ±2 cmH2O 이내 확인 (검증 기준 8.1 항)

## 4. 오리피스 단계별 기준 압력

| 단계 | 직경 | 일반 호기 압력 범위 |
|---|---|---|
| 저강도 | 4.0 mm | 10~15 cmH2O |
| 중강도 | 3.0 mm | 15~25 cmH2O |
| 고강도 | 2.0 mm | 25~35 cmH2O |

각 오리피스 10회 반복 측정 편차 ±1 cmH2O 이내일 것.
